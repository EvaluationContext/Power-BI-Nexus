# Contribute

Help us build the most comprehensive Power BI knowledge hub! Power BI Nexus is a community-driven project, and we welcome contributions from everyone.

## How You Can Contribute

- **Share Best Practices:** Have a proven technique or pattern that improved your Power BI solutions?
- **Suggest Resources**: Know a great blog, video, or tool that should be featured?
- **Improve Documentation**: Found a typo? Have a clearer explanation? See something outdated?
- **Curate Learning Tracks**: Help develop and refine our structured learning paths.

## Contribution Guidelines

1. **Fork the repository** on GitHub
2. **Make your changes** in a new branch
3. **Submit a pull request** with a clear description
4. **Engage in review** - we'll discuss and refine together
