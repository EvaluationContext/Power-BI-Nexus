# Intermediate Track

Take your Power BI skills to the next level with advanced DAX, optimized data models, and professional report design.

## Prerequisites

- Completed Beginner Track or equivalent experience
- Comfortable with basic DAX measures
- Understanding of data relationships

## Learning Path

### 1. Advanced Data Modeling

- Implementing proper star schemas
- Role-playing dimensions
- Handling many-to-many relationships
- Data minimization techniques

**Resources:**
- [Star schema design](https://learn.microsoft.com/en-us/power-bi/guidance/star-schema)
- [SQLBI: Data modeling](https://www.sqlbi.com/topics/data-modeling/)

### 2. Advanced DAX Patterns

- Context transition and row context
- Time intelligence functions
- CALCULATE and filter context
- Using variables effectively
- Iterator functions (SUMX, FILTER)

**Resources:**
- [SQLBI: DAX Patterns](https://www.daxpatterns.com/)
- [The Definitive Guide to DAX](https://www.sqlbi.com/books/the-definitive-guide-to-dax-2nd-edition/)

### 3. Power Query Mastery

- Query folding and optimization
- Custom functions and parameters
- Advanced transformations
- Incremental refresh setup

**Resources:**
- [Power Query folding](https://learn.microsoft.com/en-us/power-bi/guidance/power-query-folding)
- [Incremental refresh](https://learn.microsoft.com/en-us/power-bi/connect-data/incremental-refresh-configure)

### 4. Performance Optimization

- Using Performance Analyzer
- Introduction to DAX Studio
- Optimizing data model size
- Query optimization techniques

**Resources:**
- [Performance Analyzer](https://learn.microsoft.com/en-us/power-bi/create-reports/performance-analyzer)
- [DAX Studio](https://daxstudio.org/)

### 5. Professional Report Design

- Report themes and branding
- Accessibility best practices
- UX principles for dashboards
- Mobile-optimized layouts

**Resources:**
- [Report themes](https://learn.microsoft.com/en-us/power-bi/create-reports/desktop-report-themes)
- [Accessibility in Power BI](https://learn.microsoft.com/en-us/power-bi/create-reports/desktop-accessibility)

### 6. Security Implementation

- Row-Level Security (RLS)
- Object-Level Security (OLS)
- Dynamic security patterns

**Resources:**
- [Row-level security](https://learn.microsoft.com/en-us/fabric/security/service-admin-row-level-security)

## Next Steps

Ready for enterprise-level Power BI? Continue to the [Advanced Track](advanced.md) for deployment pipelines, architecture, and governance!
