# Beginner Track

Welcome to Power BI! This track will help you build a solid foundation in business intelligence and reporting.

## Prerequisites

- Basic understanding of data and spreadsheets
- Power BI Desktop installed ([Download here](https://powerbi.microsoft.com/desktop/))

## Learning Path

### 1. Introduction to Power BI

- What is Power BI and the Microsoft Fabric ecosystem
- Understanding the Power BI components (Desktop, Service, Mobile)
- The report development workflow

**Resources:**
- [Microsoft Learn: Get started with Power BI](https://learn.microsoft.com/en-us/training/modules/get-started-power-bi/)

### 2. Connecting to Data

- Common data sources (Excel, CSV, SQL Server)
- Using Power Query for basic transformations
- Understanding data types

**Resources:**
- [Microsoft Learn: Get data in Power BI](https://learn.microsoft.com/en-us/training/modules/get-data/)

### 3. Data Modeling Basics

- Creating relationships between tables
- Understanding star schema fundamentals
- Primary and foreign keys

**Resources:**
- [Star Schema basics](https://learn.microsoft.com/en-us/power-bi/guidance/star-schema)

### 4. Creating Visualizations

- Common visualization types and when to use them
- Formatting and customizing visuals
- Using slicers and filters

**Resources:**
- [Microsoft Learn: Visualizations in Power BI](https://learn.microsoft.com/en-us/training/modules/visuals-in-power-bi/)

### 5. Basic DAX Measures

- Understanding measures vs calculated columns
- Basic aggregations (SUM, COUNT, AVERAGE)
- Introduction to CALCULATE

**Resources:**
- [DAX Guide: Getting started](https://dax.guide/)
- [SQLBI: Introducing DAX](https://www.sqlbi.com/guides/dax/)

### 6. Publishing and Sharing

- Publishing to Power BI Service
- Creating simple dashboards
- Sharing reports with colleagues

**Resources:**
- [Microsoft Learn: Publish and share](https://learn.microsoft.com/en-us/training/modules/publish-share-power-bi/)

## Next Steps

Ready to level up? Move on to the [Intermediate Track](intermediate .md) to master advanced DAX and data modeling!
