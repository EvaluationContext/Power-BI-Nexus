# Advanced Track

Master enterprise-level Power BI development, deployment, and governance for large-scale implementations.

## Prerequisites

- Completed Intermediate Track or equivalent experience
- Experience with production Power BI environments
- Understanding of DevOps concepts is helpful

## Learning Path

### 1. Enterprise Data Architecture

- Composite models and aggregations
- Direct Lake vs DirectQuery strategies
- Data warehouse integration patterns
- Multi-tenant architectures

**Resources:**
- [Composite model guidance](https://learn.microsoft.com/en-us/power-bi/guidance/composite-model-guidance)
- [DirectQuery guidance](https://learn.microsoft.com/en-us/power-bi/guidance/directquery-model-guidance)

### 2. Advanced DAX Optimization

- Query plan analysis with DAX Studio
- Variables and materialization
- Understanding storage engine vs formula engine
- Vertipaq Analyzer for compression optimization

**Resources:**
- [SQLBI: Optimizing DAX](https://www.sqlbi.com/topics/dax-optimization/)
- [DAX internals](https://docs.sqlbi.com/dax-internals/)

### 3. External Tools Ecosystem

- Tabular Editor for model development
- Calculation groups
- TMDL (Tabular Model Definition Language)
- Best Practice Analyzer

**Resources:**
- [Tabular Editor](https://tabulareditor.com/)
- [Calculation groups](https://learn.microsoft.com/en-us/analysis-services/tabular-models/calculation-groups)

### 4. ALM and DevOps

- Power BI Projects (PBIP) and source control
- Deployment pipelines
- CI/CD automation
- Git integration strategies

**Resources:**
- [Power BI Projects](https://learn.microsoft.com/en-us/power-bi/developer/projects/projects-overview)
- [Deployment pipelines](https://learn.microsoft.com/en-us/power-bi/create-reports/deployment-pipelines-overview)

### 5. Governance and Administration

- Workspace management and capacity planning
- Monitoring with Capacity Metrics App
- Sensitivity labels and data classification
- Power BI REST API automation

**Resources:**
- [Capacity Metrics App](https://learn.microsoft.com/en-us/fabric/enterprise/capacity-metrics-app)
- [Power BI REST API](https://learn.microsoft.com/en-us/rest/api/power-bi/)
- [Microsoft Fabric REST API](https://learn.microsoft.com/en-us/rest/api/fabric/articles/)

### 6. Embedding and Custom Solutions

- Power BI Embedded architecture
- Embed for your customers vs organization
- Custom visuals development (optional)

**Resources:**
- [Power BI Embedded](https://learn.microsoft.com/en-us/power-bi/developer/embedded/embedded-analytics-power-bi)

### 7. Advanced Security Patterns

- Dynamic RLS with Azure AD
- Service principal authentication
- Managed identities for data sources

**Resources:**
- [Service principals](https://learn.microsoft.com/en-us/power-bi/enterprise/service-premium-service-principal)
