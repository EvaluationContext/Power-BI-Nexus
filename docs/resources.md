# Resources

Resources to help you get up to speed with Power BI.

??? info "RSS Feeds"

    Stay up-to-date with the latest Power BI and Microsoft Fabric content from official sources and leading community creators. Subscribe to these RSS feeds in your favorite RSS reader.

    Copy the RSS feed URL and add it to your RSS reader of choice:

    - **Browser Extensions:** Feedly, Inoreader
    - **Desktop Apps:** NetNewsWire, Feedbro
    - **Mobile Apps:** Reeder, Feedly Mobile
    - **Email:** Blogtrottr, Kill the Newsletter!

    ??? tip "Managing RSS Feeds"

        1. **Organize by category** - Group feeds by Official, Community, Video, etc.
        2. **Set reading schedules** - Check feeds daily or weekly depending on content volume
        3. **Use filters** - Set up keyword alerts for topics you're most interested in
        4. **Save articles** - Use read-it-later services like Pocket or Instapaper
        5. **Share discoveries** - Consider contributing valuable content back to Power BI Nexus!
