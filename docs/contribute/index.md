# Contribute

Help us build the most comprehensive Power BI knowledge hub! 

Power BI Nexus is a community-driven project, and we welcome contributions from everyone. Please read this guide thoroughly before contributing.

## Ways to Contribute

We use GitHub issues to manage all contributions. Please use the appropriate issue template for your contribution:

### :material-link-plus: Suggest a Resource

Found a valuable blog, tool, video, or documentation that should be featured?

[Create a resource suggestion :octicons-arrow-right-24:](https://github.com/EvaluationContext/Power-BI-Nexus/issues/new?template=suggest-resource.yml){ .md-button .md-button--primary }

### :material-file-document-edit: Improve Best Practices

Have a proven technique or updated guidance backed by authoritative sources?

[Suggest best practice improvement :octicons-arrow-right-24:](https://github.com/EvaluationContext/Power-BI-Nexus/issues/new?template=improve-best-practices.yml){ .md-button .md-button--primary }

### :material-bug: Report an Issue

Found a typo, broken link, or outdated information?

[Report an issue :octicons-arrow-right-24:](https://github.com/EvaluationContext/Power-BI-Nexus/issues/new?template=report-issue.yml){ .md-button .md-button--primary }
